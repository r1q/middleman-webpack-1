#  Middleman Starter Project

## Features
 
* Sass (Node)
* Webpack
* ES6 (ES2015)
* Ruby Slim
* Browser Sync (live reload etc)

## Installation

* Create new project using this template 
 
  ```
  middleman init my-project --template=tyom/middleman-webpack#mojular
  ```
* Install NPM packages 

  ```
  cd my project
  npm install
  ```

## Usage

* Run dev server

  ```
  middleman server
  ```
  
* Build static project

  ```
  middleman build
  ```
